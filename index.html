Welcome in the world of SQDev

SQDev is a plugin for eclipse that will teach it how to support SQF. Therefore all the features of a modern IDE will be available for you as soon as you have installed the plugin.

Status + Information

Currently v0.7.4 is the most recent version.
The plugin, its features, usage and much more is documented in the Manual.

Troubles and bugs

If you have any troubles concerning this plugin you can contact me @Krzmbrzl and I will see what I can do for you. If you encounter any bugs please create an issue to let me know about it so I can fix it.

Feature requests

I am open for all kind of feature requests. If you have one you can as well create an issue and I will see what I can do
